## Installation
```
git clone https://github.com/DebNation/bunkr-downloader.git
cd bunkr-downloader
python3 -m venv env
# for Linux/Mac
source env/bin/activate
#
pip install -r requirements.txt
```
## Usage
- Create a URLS.txt file inside the project directory, put yours urls in it
- then run
```
python main.py
```
